Introduction

This project from 42 School aims to enhance your understanding of Docker by creating virtualized environments for various applications.

Key Definitions

What is Docker?

Docker is an open platform for developing, shipping, and running applications in isolated environments called containers.

What is Docker Compose?

A tool for defining and running multi-container Docker applications using a YAML configuration file.

What is a Dockerfile?

A text document containing all the commands to assemble an image.

Steps to Follow

  • Start with Nginx: Display an index.html page on http://localhost:80.
  • Enable SSL: Display the page on http://localhost:443 with SSL.
  • Move to WordPress: Set up a WordPress container with Docker Compose.
  • Set up MariaDB: Create a database to store WordPress data.
  • Test your containers: Import pre-built images and ensure functionality.

Useful Docker Commands

Command Description
docker build Create a Docker image from a Dockerfile.
docker run Run a container based on an image.
docker ps List all running containers.
docker-compose up Start and build multi-container applications.
